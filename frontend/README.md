# Frontend

Фронтенд учебного проекта **«Система учёта футбольных турниров»**.

## Структура

```text
app/
  dashboard/
  login/
  matches/
  standings/
  teams/
  tournaments/
components/
  access-guard.tsx
  auth-provider.tsx
  dashboard-overview.tsx
  dashboard-role-panels.tsx
  matches-client.tsx
  mock-api.ts
  mock-data.ts
  standings-client.tsx
  teams-client.tsx
  tournaments-client.tsx
```

## Запуск

```bash
npm install
npm run dev
```
