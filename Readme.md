# Football Tournament System

Монорепозиторий учебного проекта **«Система учета футбольных турниров»**.

## Структура

```text
frontend/       # Next.js frontend
docs-stub/      # TypeDoc-стаб для документации
```

## Быстрый старт

```bash
npm install
npm run dev
```

Frontend будет доступен по адресу `http://localhost:3000`.

## Полезные команды

```bash
npm run dev
npm run build
npm run lint
npm run docs
```

## Где что находится

- `apps/frontend` — пользовательский интерфейс на `Next.js + React + TypeScript`
- `tools/docs-stub` — отдельный пакет для генерации документации через `TypeDoc`
